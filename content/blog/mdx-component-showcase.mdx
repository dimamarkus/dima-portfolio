## Code Blocks

Our CodeBlock component provides syntax highlighting and a copy button:

```javascript
// This is a JavaScript function
function greeting(name) {
  return `Hello, ${name}!`;
}

// ES6 arrow function
const sum = (a, b) => a + b;

// Using the function
console.log(greeting("World"));
```
