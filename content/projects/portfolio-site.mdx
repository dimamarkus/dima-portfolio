# Portfolio Website Project

This project is a modern, responsive portfolio website built to showcase my work and skills as a developer. It features a clean, minimalist design with a focus on content and user experience.

## Project Goals

The main goals for this portfolio website were:

1. **Create a modern, responsive design** that works well on all devices
2. **Showcase my projects and skills** in a clean, organized way
3. **Implement a hybrid content management system** using both WordPress and MDX
4. **Optimize for performance and SEO** to ensure good visibility and user experience

## Technical Implementation

The website is built with Next.js 15 using the App Router, which provides several benefits:

- **Server Components** for improved performance and SEO
- **Server Actions** for form handling and data mutations
- **Static Site Generation** for fast page loads

The styling is done using TailwindCSS, which allows for rapid development and consistent design. The content is managed through a combination of WordPress (for dynamic content) and MDX files (for version-controlled content).

## Key Features

- **Responsive design** that works on all devices
- **Dark/light mode toggle** for user preference
- **Project showcase** with filtering capabilities
- **Blog section** with MDX support for rich content
- **Contact form** with server-side validation
- **SEO optimization** with Next.js metadata API

## Challenges and Solutions

One of the main challenges was implementing a hybrid content management system that could pull data from both WordPress and local MDX files. This was solved by creating a custom utility that normalizes data from both sources and provides a unified interface for the components.

## Results

The final product is a fast, responsive portfolio website that effectively showcases my work and skills. The site achieves perfect Lighthouse scores for performance, accessibility, best practices, and SEO.
